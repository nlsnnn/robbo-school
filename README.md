# Robbo School

## Описание

Система управления курсами, разработанная в рамках тестового задания. Проект представляет собой фуллстек приложение, реализующее функционал создания, просмотра, редактирования и удаления учебных курсов.

## Стек технологий

### Backend
- **Go** (Golang) - основной язык разработки
- **Chi** - легковесный и быстрый HTTP-роутер
- **PostgreSQL** - реляционная база данных
- **pgx** & **sqlx** - драйвер и инструменты для работы с БД
- **Cleanenv** - управление конфигурацией

### Frontend
- **React** - библиотека для пользовательского интерфейса
- **Vite** - быстрый сборщик и dev-сервер
- **Tailwind CSS** - CSS фреймворк
- **Axios** - HTTP клиент для взаимодействия с API

### Инфраструктура
- **Docker** & **Docker Compose** - контейнеризация и оркестрация сервисов
- **Nginx** - веб-сервер для раздачи статики фронтенда

## Быстрый старт

1. Клонировать репозиторий:

   ```bash
   git clone https://github.com/nlsnnn/robbo-school.git
   cd robbo-school
   ```

2. Запустить проект через Docker Compose:

   ```bash
   docker-compose up --build
   ```

3. Открыть в браузере:
   - **Frontend**: http://localhost:3000
   - **API**: http://localhost:8080

## Эндпоинты API

| Метод  | Маршрут          | Описание                                      |
| ------ | ---------------- | --------------------------------------------- |
| GET    | `/courses`       | Получить список всех курсов                   |
| GET    | `/courses/{id}`  | Получить детальную информацию о курсе по ID   |
| POST   | `/courses`       | Создать новый курс                            |
| PUT    | `/courses/{id}`  | Обновить существующий курс                    |
| DELETE | `/courses/{id}`  | Удалить курс                                  |

## Локальная разработка

### Backend

1. Перейдите в директорию `api`:
   ```bash
   cd api
   ```

2. Установите зависимости:
   ```bash
   go mod download
   ```

3. Запустите базу данных и миграции (используя docker-compose):
   ```bash
   docker-compose up -d db migrations
   ```

4. Запустите сервер:
   ```bash
   go run cmd/api/main.go
   ```

### Frontend

1. Перейдите в директорию `frontend`:
   ```bash
   cd frontend
   ```

2. Установите зависимости:
   ```bash
   npm i
   ```

3. Запустите режим разработки:
   ```bash
   npm run dev
   ```
   Приложение будет доступно по адресу http://localhost:3000
